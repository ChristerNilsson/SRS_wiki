    Lathund¶

    1. Gå in på redigeringssidan
    2. Klicka på Wiki
    3. Klicka på sidan du vill ändra eller New Page
    4. Klicka på Edit
    5. Redigera
    6. Klicka på Preview därefter Write.
    7. Klicka på Save när du är nöjd med sidan
    8. Klicka på Actions för att se när sidan publicerats. Alla gula lampor ska bli gröna. Det tar någon minut
    9. Se publik sida
    10. Kom ihåg att använda små bokstäver för alla sidnamn. De blir filnamn och Linux skiljer på versaler och gemener.

    Lathund för Markdown¶

    För dig som vill skriva enkelt på klubbens wiki

    Markdown är ett lätt sätt att skriva text som blir snygg på webben. Man använder några enkla tecken för rubriker, listor, länkar osv.

    Tips¶

    • Använd # rubriker för att strukturera texten
    • Ha tom rad mellan stycken
    • Bilder är enklast att lägga in mha drag-and-drop
    • PDF-filer och andra dokument hanteras enklast med att länka till dem på andra servrar
    • Håll det enkelt – man behöver inte kunna alla finesser
